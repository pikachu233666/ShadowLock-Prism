ShadowLock Prism - Yuxin Liu

The design is inspired by traditional puzzles: Tangram represents 2D geometric composition, while Kongming Lock represents complex 3D structural interlocking. This project combines both ideas by extending 2D spatial reasoning into a fully 3D environment.

Rules
  • Every occupied cell must match the glowing target volume.
  • No overlap is allowed.
  • No floating cells: each cell must sit on the ground or another occupied cell.
  • You win only when the full target volume is covered exactly once.
